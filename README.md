# Form & Layout Studio

This project is a static HTML and CSS page built for layout practice. It includes:

- A multi-section page built with flexbox
- A positioned label example using relative and absolute positioning
- A form layout with inputs, select, textarea, radio buttons, and a checkbox
- A responsive card gallery built with CSS Grid
- Box model practice with margins, padding, borders, and spacing

## Files

- `index.html` - Main page content and structure
- `styles.css` - All layout, spacing, color, and responsive styling

## How to Open

1. Open the `Static_HTML_CSS_PAGE` folder in VS Code.
2. Open `index.html`.
3. Use Live Server or open the file directly in a browser.

## What It Demonstrates

This page was designed to look like a real website rather than a demo. It uses a clean theme, visible accent colors, and simple content blocks so the layout is easy to understand.

## Notes

- No JavaScript is used.
- The page is fully static.
- The layout is responsive and adapts to smaller screens.
